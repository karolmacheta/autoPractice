# autoPractice

Cypress Playground

## Cypress Documentation:

https://www.cypress.io/

# Installation:

## Clone the repository:

```bash
$ git clone https://github.com/karolmacheta/autoPractice.git
```
## Enter the folder and run npm install

```bash
$ cd autoPractice
$ npm install
```

# Running the test:

```bash
$ npx cypress open
```
